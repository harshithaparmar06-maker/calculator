#  Basic Calculator App

A clean, modern browser-based calculator built with **HTML, CSS, python and JavaScript**.

##  Features
- All arithmetic operations: +, −, ×, ÷
- Percentage (%) and toggle sign (+/−)
- Expression display showing full calculation
- **Full keyboard support** (numbers, operators, Enter, Escape, Backspace)
- Division by zero error handling
- Floating point precision handling
- Smooth button press animations

##  Keyboard Shortcuts
| Key | Action |
|---|---|
| `0–9` | Input digits |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Escape` | Clear all |
| `Backspace` | Delete last digit |

##  How to Run
Just open `index.html` in any browser — no installation needed!


##  Tech Stack
`HTML5` `CSS3` `JavaScript (ES6)` `python`
